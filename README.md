# DMX Ecommerce (Monorepo)

Private commercial ecommerce platform for the DMX (Divik Home Clean Products) brand.

Includes:
- Storefront (React + Vite)
- Admin Panel (CRUD + Cloudinary)
- Backend (Express + MVC)
- Payments (Razorpay)
- MongoDB Atlas
- Cloudinary Storage
- JWT Auth
- Orders + Checkout

## Deployment Model

Frontend → Vercel  
Backend → Render  
DB → MongoDB Atlas  
Payments → Razorpay  
Images → Cloudinary
